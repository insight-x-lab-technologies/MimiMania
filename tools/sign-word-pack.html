Internal Word Pack Signer

Generates the JSON envelope accepted by the MimiMania installer using the same hash, payload, and signature rules as the game.

Internal tool. This file lives in tools/, outside the current GitHub Pages deployment, which publishes only src/. Do not put the private key in src/ and do not publish this page.

Input

If you load an existing envelope, the tool hashes only content and fills user_id/pack_id.

Expected PEM format: -----BEGIN PRIVATE KEY-----. If you have BEGIN EC PRIVATE KEY, convert it to PKCS#8.

Output

The signature is created over: mimimania-word-pack:v1, user_id, pack_id, and content_sha256.
